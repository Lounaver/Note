

# Python模块

## os模块

### os模块的部分常用属性（Windows）

| ***\*OS模块常用属性\**** | ***\*属性简介\****                        |
| ------------------------ | ----------------------------------------- |
| os.name                  | 返回电脑的操作系统类型（Windows返回“nt”） |
| os.curdir                | 当前工作目录即：’.’                       |
| os.pardir                | 当前工作目录的上级目录即：’..’            |
| os.sep                   | 返回路径分隔符，’\’\’                     |
| os.extsep                | 返回文件扩展名分隔符，Windows下则返回’.’  |
| os.linesep               | 返回文本文件行分隔符，即’\n\r’            |

 

### os模块常用方法

| os模块常用方法 | 方法简介 |
| :----------: | :--------: |
| os.system(command) | 系统命令行执行command命令 |
